# NoteChatRecode

NoteChatRecode 是一个基于 .NET 的聊天应用程序，包含客户端和服务器端。

## 项目结构

- **NoteChatRecode-Client**: 客户端项目，目标框架为 .NET Framework 4.7.2
- **NoteChatRecode-Server**: 服务器端项目，目标框架为 .NET 8

## 运行环境

### 客户端

- .NET Framework 4.7.2
- Visual Studio 2022

### 服务器端

- .NET 8
- Visual Studio 2022

## 构建和运行

### 客户端

1. 打开 `NoteChatRecode-Client` 解决方案。
2. 生成解决方案。
3. 运行 `NoteChatRecode-Client` 项目。

### 服务器端

1. 打开 `NoteChatRecode-Server` 解决方案。
2. 生成解决方案。
3. 运行 `NoteChatRecode-Server` 项目。

## 项目文件说明

### NoteChatRecode-Client

- `Program.cs`: 应用程序的主入口点，初始化并运行主窗体 `MainForm`。

### NoteChatRecode-Server

- `NoteChatServer.cs`: 定义服务器的基本信息和功能。
- `UserData.cs`: 用户数据管理类。

### NoteChatRecode-Common

项目的共用代码。
